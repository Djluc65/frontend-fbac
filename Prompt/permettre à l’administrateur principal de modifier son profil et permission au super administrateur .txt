11. INVITATION ADMINISTRATEUR
==================================================

Créer un système d’invitation.

Modèle AdminInvitation :

- email ;
- firstName ;
- lastName ;
- role ;
- permissions ;
- tokenHash ;
- expiresAt ;
- status ;
- invitedBy ;
- acceptedAt ;
- createdAt.

Statuts :

- pending
- accepted
- expired
- revoked

Flux :

1. le super administrateur invite un responsable ;
2. le backend crée un token sécurisé ;
3. le token brut est envoyé par email ;
4. seul le hash est conservé en base ;
5. le lien expire après 24 ou 48 heures ;
6. l’invité définit son mot de passe ;
7. le compte devient actif ;
8. l’invitation passe à accepted ;
9. un AuditLog est créé.
